## 🌟 Project Vision: "상상을 현실로, 주문을 감동으로"

MakeAWish-AI는 수제 커스텀 케이크 시장의 **주문 시각화의 어려움**과 **제작 운영의 비효율**을 해결하는 통합 AI 어시스턴트 플랫폼입니다. 

고객에게는 세상에 하나뿐인 디자인 경험을, 사장님에게는 비즈니스 운영의 여유를 드리는 **Full-Stack AI 솔루션**을 지향합니다.

---

## 🚀 Key Features

### 🎨 AI 시안 에디터 (Customer Side)
- **버터크림 화풍 보존**: 기존 케이크의 질감과 색감을 유지하며 원하는 영역만 정교하게 수정.
- **실시간 디자인 시뮬레이션**: 텍스트 명령과 참고 사진을 통해 즉석에서 케이크 시안 생성.
- **멀티모달 인터랙션**: 이미지, 마스크, 텍스트, 참고 사진을 결합한 직관적인 UI/UX.

### 💼 스마트 오더 시스템 (Shop Owner Side)
- **주문서 자동 추출**: AI 채팅을 통한 슬롯 필링(Slot-filling) 방식의 주문 정보 구조화.
- **리뷰 및 데일리 브리핑**: 매장 리뷰 요약 및 일일 매출/주문 현황 자동 생성.
- **고객 응대 AI 챗봇**: 알러지 체크 및 디자인 상담 지원.

## 🛠 Tech Stack

### Frontend & Mobile
- **React Native / Expo**: iOS & Android 통합 개발 및 크로스 플랫폼 사용자 경험 제공.
- **Canvas Interface**: 정밀한 마스킹 및 실시간 이미지 렌더링을 위한 전용 UI.

### Backend & AI Engine
- **Spring Boot**: 메인 비즈니스 로직, 데이터베이스 관리, 주문 라이프사이클 제어.
- **FastAPI (AI Engine)**: 고성능 비동기 통신을 통한 Gemini 3.1 Flash 모델 서빙.
- **Core AI**: Google Gemini 3.1 Flash (Image Edition) API.

### Infrastructure & Communication
- **RESTful API**: Spring Boot와 FastAPI 간의 효율적인 마이크로서비스 통신.
- **Image Data Pipeline**: Base64 기반의 실시간 이미지 스트리밍 기술.

## 💎 Technical Edge (Why MakeAWish-AI?)

1.  **초고속 응답 (100x Speed Up)**: 로컬 모델 대비 **약 100배 빠른 속도(3~5초)**로 실시간 모바일 환경에 최적화.
2.  **질감의 리얼리티**: 실사 사진이 아닌 '케이크 크림' 특유의 질감을 99% 재현하는 프롬프트 엔지니어링.
3.  **심리스한 통합 아키텍처**: 모바일(React Native) - 서버(Spring Boot) - AI(FastAPI)로 이어지는 안정적인 데이터 흐름.
